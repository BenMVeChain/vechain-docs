# Activities

{% tabs %}

{% tab title="Mobile" %} 

To watch your history you should:

- Go to your Dashboard
- Tap the History button
- You'll see a list of activies (some of them are onchain, some are don't)
- Tap one of them to read more
- In the details you'll see:
  - From and To Address
  - Datetime of creation
  - Value (which and how many tokens)
  - Transaction ID (you can also copy it in your clipboard)
  - Blocknumber
  - Network
  - Button to read details for that transaction in the blockexplorer
